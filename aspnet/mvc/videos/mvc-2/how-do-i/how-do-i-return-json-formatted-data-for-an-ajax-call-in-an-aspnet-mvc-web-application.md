<a name="how-do-i-return-json-formatted-data-for-an-ajax-call-in-an-aspnet-mvc-web-application"></a>实现： 如何返回 JSON 格式的 ASP.NET MVC Web 应用程序中的 AJAX 调用的数据？
====================
通过[Chris Pels](https://twitter.com/chrispels)

在本视频中 Chris Pels 演示如何使用 JsonResult 和 Json 类返回类的实例为 JSON 格式的数据。 首先，创建一个示例 MVC web 应用程序和客户信息添加一个控制器和关联的视图。 然后为数据模型的一部分创建一个简单的 customer 类。 GetCustomer() 函数然后添加到 customer 控制器来检索 JSON 格式中的单个客户的数据。 最后，脚本将添加到客户视图，使用 jQuery 进行 GetCustomer() AJAX 调用以检索 JSON 格式的客户数据
