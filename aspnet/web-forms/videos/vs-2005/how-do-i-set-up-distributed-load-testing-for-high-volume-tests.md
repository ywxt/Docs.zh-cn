Visual Studio Team System 可以设置分布式负载测试所创建的测试"装置"包含相同或其他远程计算机上的多个代理和一台远程计算机上的控制器。 此视频将引导您完成安装控制器和代理，然后配置测试用例，以使用远程控制器运行。 正如您将看到，Visual Studio Team System 可以非常容易设置分布式负载测试以进行大规模自动测试。
