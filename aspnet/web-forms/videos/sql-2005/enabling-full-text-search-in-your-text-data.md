<a name="enabling-full-text-search-in-your-text-data"></a>启用文本数据中的全文搜索
====================
by [Microsoft](https://github.com/microsoft)

全文搜索，可将文本-甚至 Microsoft Word 文档或其他文件格式的大部分内容保存到你的数据库并执行复杂的查询基于此文本数据，但它应该优异的性能。 本课程演示如何设置全文搜索设置 SQL Server Express 中的细微差别，特别注意。 然后我们移动到创建全文索引、 配置索引填充设置，以及执行演示的 CONTAINS 谓词和其许多变体的强大的查询。
